# Serendipity for JetBrains

Elegant, minimal, and clean color palette for your tools.

See other interfaces at the [official website](https://www.michaelandreuzza.com/vscode/serendipity/).

## Available themes

- **Midnight** — dark
- **Morning** — light
- **Sunset** — dark

## Installation

See `INSTALL.md` for import steps.

Available files: `Serendipity Midnight.icls`, `Serendipity Morning.icls`, `Serendipity Sunset.icls`, `INSTALL.md`.
